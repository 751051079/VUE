## Version：1.0.0
### 提交代码
```

//添加到本地仓库
git add .

//提交代码
git commit -m ‘提交信息’

//推送
git push

//拉取
git pull
```

### 创建项目

```
//项目安装依赖
yarn install

//项目运行
yarn serve

//项目打包
yarn build

//检查项目配置
yarn lint

```
